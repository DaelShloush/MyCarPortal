# MyCarPortal — מסמך פיצ'רים מפורט

## סקירה כללית

MyCarPortal מציעה שני מוצרים בפלטפורמה אחת:
1. **כלי חיפוש רכב** — חיפוש לפי מספר רישוי עם דירוג סיכון
2. **כלי ניהול רכב אישי** — דשבורד, תזכורות, מסמכים

---

## 1. חיפוש רכב לפי מספר רישוי

### 1.1 שדה חיפוש

| פרט | פירוט |
|---|---|
| **מיקום** | מרכז עמוד הנחיתה + Navbar קבוע |
| **קלט** | 5-8 ספרות בלבד |
| **ולידציה** | regex `^\d{5,8}$`, הודעת שגיאה בעברית |
| **אפשרות נוספת** | כפתור מצלמה → OCR עם Claude Vision (Should Have) |
| **חיפוש מהיר** | autocomplete מהיסטוריית החיפושים של המשתמש |
| **UX** | Enter או לחיצה על כפתור "חפש", Loading skeleton עד לתוצאה |

### 1.2 עמוד תוצאות חיפוש — Single Page

כל המידע מוצג **בעמוד אחד** עם גלילה, מחולק לסקשנים עם כותרות ברורות. לא tabs.

#### Header (חלק עליון קבוע)
- **תמונת רכב:** תמונת CGI מ-imagin.studio (לפי יצרן, דגם, שנה, צבע)
- **לוגו יצרן:** SVG מ-avto-dev/vehicle-logotypes CDN
- **שם הרכב:** יצרן + דגם מסחרי + שנת ייצור (למשל: "טויוטה COROLLA 2019")
- **יד:** מספר הבעלות הפרטית (למשל: "יד 3" — חישוב מהיסטוריית בעלויות, ללא סוחרים/ליסינג)
- **Risk Badge:** עיגול צבעוני עם הציון (0-100) ומסר קצר
- **כפתורי פעולה:**
  - ⭐ הוסף למועדפים (דורש הרשמה)
  - 📤 שתף (WhatsApp / העתק קישור)
  - 📄 הורד PDF (פרמיום)
  - 🚗 הוסף לרכבים שלי (דורש הרשמה)

#### תמונת רכב (imagin.studio)
- **URL pattern:** `https://cdn.imagin.studio/getimage?customer=KEY&make=MAKE&modelFamily=MODEL&modelYear=YEAR&paintId=COLOR`
- **מיפוי:** `tozeret_nm` → make, `kinuy_mishari` → model, `shnat_yitzur` → year, `tzeva_rechev` → paintId
- **Free tier:** 400px + watermark, ללא CORS
- **Paid:** מ-$99/חודש, ללא watermark, רזולוציה גבוהה
- **Fallback:** אם אין תמונה (דגם לא מוכר) → placeholder כללי

#### חישוב "יד"
- **לוגיקה:** ספירת בעלויות מסוג "פרטי" בלבד מתוך ownership history

### 1.3 מה קורה כשרכב לא נמצא

סדר בדיקה:
1. Main registry → נמצא? ✅ הצג
2. Inactive vehicles → נמצא? הצג + "⚠️ הרכב אינו פעיל"
3. Decommissioned → נמצא? הצג + "🔴 הרכב נגרט בתאריך X"
4. Personal import → נמצא? הצג (פחות שדות)
5. Public vehicles → נמצא? הצג + "ℹ️ רכב ציבורי"
6. לא נמצא בכלל → "לא נמצא רכב עם מספר רישוי X. בדוק שהמספר הוקלד נכון."

## 2. דירוג סיכון (Risk Score)

### 2.1 אלגוריתם

חישוב בזמן אמת. ציון 0-100 (0 = מצוין, 100 = מסוכן).

#### פרמטרים

**בעלויות (20 נקודות)**
```
1-2 בעלים  → 0
3 בעלים    → 6
4 בעלים    → 12
5 בעלים    → 16
6+ בעלים   → 20
```

**תדירות החלפה (10 נקודות)**
```
לכל בעלים שהחזיק < 6 חודשים: +3 (מקס 10)
```

**גיל הרכב (15 נקודות)**
```
0-3 שנים   → 0
4-7 שנים   → 5
8-14 שנים  → 10
15+ שנים   → 15
```

**סטטוס טסט (15 נקודות)**
```
בתוקף > 6 חודשים   → 0
בתוקף 3-6 חודשים   → 4
בתוקף < 3 חודשים   → 8
פג תוקף             → 15
```

**קילומטראז' (15 נקודות)**
```
ק"מ/שנה:
8,000-20,000 (סביר)    → 0
20,001-30,000 (גבוה)    → 8
> 30,000 (גבוה מאוד)    → 12
< 5,000 (חשוד)          → 15  // ייתכן איפוס מונה
לא זמין                 → 7   // ציון ביניים
```

**שינוי מבנה (10 נקודות)**
```
לא → 0
כן → 10
```

**ריקולים פתוחים (10 נקודות)**
```
0 ריקולים  → 0
1 ריקול    → 5
2+ ריקולים → 10
```

**סוג בעלות (5 נקודות)**
```
פרטי      → 0
חברה      → 1
החכר      → 2
השכרה     → 5
```

## 4. דשבורד "הרכבים שלי"

### 4.1 תצוגה
- רשימת כרטיסי רכב (1 בחינם, עד 3 בפרמיום)
- כל כרטיס: שם/כינוי, יצרן+דגם, שנה, countdown לטסט, countdown לביטוח
- **Alert bar** למעלה אם יש תזכורת שקרבה (< 30 יום)
- כפתור "הוסף רכב" (נחסם אם הגיע למגבלה)


### 4.3 הוספת רכב
1. הזנת מספר רישוי
2. שליפה אוטומטית מ-data.gov.il
3. המשתמש מוסיף: כינוי, תאריך תום ביטוח
4. נשמר ב-DB עם cache של הנתונים
5. תזכורות טסט נוצרות אוטומטית מ-`tokef_dt`

### 4.4 סנכרון נתונים
- בכל כניסה לדשבורד: בודק אם `last_synced_at` > 24 שעות
- אם כן → שולף מחדש מה-API ומעדכן

## 8. OCR — צילום לוחית רישוי (Should Have)

### 8.1 Flow
1. משתמש לוחץ על אייקון מצלמה בשדה החיפוש
2. נפתחת מצלמה (Web API: `navigator.mediaDevices.getUserMedia`)
3. צילום → שליחה ל-API Route → Claude Haiku Vision
4. Claude מחלץ מספר רישוי → מוחזר לשדה חיפוש
5. חיפוש אוטומטי

### 8.2 Prompt ל-Claude
```
Extract the Israeli license plate number from this image.
Return ONLY the digits, no dashes or spaces.
If no license plate is found, return "NOT_FOUND".
```

### 8.3 מגבלות
- עובד רק ב-HTTPS (דרישת browser לגישה למצלמה)
- דורש הרשאת מצלמה מהמשתמש
- עלות: ~$0.001 לקריאה (Claude Haiku)
